# Data-Analyst-Role-ECE-Corporate-Partners-Mihir-Samanyu-Talamudipi
Survey and analysis of Corporate Partnership programs across the top 20 ECE departments in the U.S. This project compiles data on partner companies, technology sectors, membership tiers, fees, and benefits. Includes dataset, analysis notebooks, and visualizations.

This project investigates the Corporate Partnerships programs at the top 20 Electrical and Computer Engineering (ECE) departments in the United States. The goal is to understand how leading universities collaborate with industry through structured partnership programs.

The analysis includes:

- Identification of corporate partners for each university

- Classification of technology sectors represented

- Comparison of membership tiers, associated fees, and benefits

- Visual exploration of trends across institutions

All findings are organized into a structured dataset and presented with data visualizations for clarity. To run the python script to see the visualizations, make sure everything is in the same folder. As for the report and dataset - download the raw files to open them.
